# 🌤️ Weather Dashboard

A beautiful, responsive weather application that fetches real-time weather data from the Open-Meteo API.

## Features

✨ **What it does:**
- 🔍 Search weather by city name
- 🌡️ Display current temperature, humidity, and wind speed
- 📅 Show 7-day weather forecast
- 🎨 Beautiful and responsive design
- 📱 Works on desktop, tablet, and mobile

## How to Use

1. Visit: https://YOUR-USERNAME.github.io/weather-dashboard/
2. Enter a city name in the search box
3. Click "Search"
4. View current weather and 7-day forecast

## Technologies Used

- **React** - UI framework
- **Open-Meteo API** - Weather data (free, no API key needed)
- **CSS3** - Styling
- **JavaScript** - Logic

## API

This project uses the free [Open-Meteo API](https://open-meteo.com/):
- No API key required
- Free unlimited requests
- Global weather coverage
- Forecast data included
